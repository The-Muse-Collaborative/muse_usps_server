muse_usps_server package¶

Module contents¶

Simple flask server allowing AJAX requests to validate addresses using the USPS API.

muse_usps_server.validate()¶

Validates an address that comes in as JSON data.

See muse_usps.validate documentation for request structure. Any extra fields are ignored. Returns a 400 error if the request was valid, but an error was reported by the USPS API. The 400 response will include an “error” field with the error message returned by the USPS API. All other backend errors will result in a 500 error with no response.
